# HOT-PLACE

graduation project, HOT PLACE

'HOT PLACE' is Diary Application that provides Real-Time user place informaion.

copyright ⓒ 2019 lee-su-hyeong (Chungbuk National Univ. software) all rights reserved

* Framework : Android Studio
* Language : JAVA
* Database : Firebase
* API : Google Maps API, OpeanWeatherMap API (Current weather data), Naver Maps API (Real-Time Traffic Info)

* * *
## 사용자 요구사항
* 회원 관리
  * 사용자는 이메일로 회원 가입을 할 수 있다.
  * 가입이 되어있는 사용자는 로그인 및 로그아웃, 회원탈퇴를 할 수 있다.
  * 사용자는 로그인 후 회원 정보를 수정할 수 있다. (비밀번호 수정 및 프로필 사진 수정)
  * 사용자는 자신의 회원 정보를 조회할 수 있다.
  
* 정보 제공
  * 사용자는 메인 화면에서 자신의 위치에 따른 날씨 정보를 조회할 수 있다.
  * 사용자는 메인 화면에서 자신의 위치에 따른 교통 정보를 조회할 수 있다.
  
* 그 외 사용자 기능
  * 사용자는 자신의 위치 외에 날씨 정보 혹은 교통 정보를 알고 싶은 지역을 검색하여 정보를 조회할 수 있다.
  * 사용자는 자신의 위치에 메모를 작성하고 저장할 수 있다.
  * 사용자는 자신이 작성한 메모의 공개 범위를 전체 공개, 친구 공개, 비공개로 설정할 수 있다.
  * 사용자는 자신의 위치를 공개 혹은 비공개로 설정하여 친구인 사용자에게 공개 혹은 비공개할 수 있다.
  * 사용자는 친구를 맺길 원하는 사용자의 이메일을 검색하여 친구 신청을 할 수 있다.
  * 친구 신청을 받은 사용자는 친구 신청을 승낙, 거절, 무응답할 수 있다.

> Main Activity
1. 사용자는 자신의 이메일로 회원가입을 할 수 있으며, 회원가입한 이메일로 로그인할 수 있다.
2. 사용자는 자신의 위치를 구글 지도로 확인 할 수 있으며, 자신의 위치에 따른 날씨 정보와 교통 정보를 확인 할 수 있다.
3. 사용자는 자신의 위치 외에 친구가 저장한 메모와 메모의 위치를 확인 할 수 있다.

> Search Activity
1. 사용자는 날씨 정보와 교통 정보를 알고 싶은 지역을 검색하여 정보를 얻을 수 있다.
2. 지역 검색은 특별시/광역시/도, 시/군/구, 읍/면/리로 나누어져 있다.

2. 사용자는 자신이 저장한 메모를 확인할 수 있다.

2. 사용자는 자신의 메모 공개 범위를 전체 공개, 친구 공개, 비공개로 설정할 수 있다.
3. 사용자는 자신의 위치를 친구들에게 공개 혹은 비공개로 설정할 수 있다.
